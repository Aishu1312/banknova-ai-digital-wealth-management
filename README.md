# BankNova AI — Digital Wealth Management

BankNova AI is an intelligent Wealth OS designed to help users manage their personal finances, track spending, set financial goals, and get AI-driven investment advice.

## Features

- **Home Dashboard**: Get a snapshot of your total portfolio, monthly spend, and holding breakdowns.
- **NOVA Chat**: Interact with an AI wealth advisor to understand your spending, goals, and portfolio.
- **Spend Analysis**: Visual breakdown of your monthly spending categories and recent transactions.
- **Invest**: Take a risk assessment quiz and receive tailored investment recommendations including SIPs, FDs, and Mutual Funds.
- **Goal Planning**: Set custom financial goals, calculate required SIPs, and track your progress.
- **Alerts**: Receive notifications about unusual spending, SIP opportunities, and goal milestones.

## Technologies Used

- [Streamlit](https://streamlit.io/)
- [Pandas](https://pandas.pydata.org/)
- [Plotly](https://plotly.com/python/)

## Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Aishu1312/banknova-ai-digital-wealth-management.git
   cd banknova-ai-digital-wealth-management
   ```

2. Create a virtual environment (optional but recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
   ```

3. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Run the Streamlit app:
   ```bash
   streamlit run app.py
   ```
