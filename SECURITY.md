# Security Policy

## Supported Versions

Currently, only the latest version of BankNova AI is actively supported for security updates.

## Reporting a Vulnerability

If you discover a security vulnerability within BankNova AI, please do not disclose it publicly. Instead, contact the repository maintainers privately via their GitHub profiles.

We will endeavor to respond to your report as quickly as possible and provide an update on when a fix is expected to be released.
